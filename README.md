# Intelligent SQLite-Based Pharmacy Management System

This repository contains a lightweight pharmacy management system developed using Python and SQLite.

## Features
- Inventory management
- Sales processing
- Expiry alerts
- Low stock alerts
- Demand forecasting (Moving Average, Linear Regression, Random Forest)

## How to Run
1. Install Python 3.x
2. Install requirements:
   pip install pandas scikit-learn matplotlib
3. Run:
   python pharmacy.py
